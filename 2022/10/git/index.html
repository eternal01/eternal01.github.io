    Git配置 #

    Git常用命令 #

    初始化仓库

    git init

    添加本地修改至暂存区

    git add .   #添加所有修改
git add -A  #添加所有修改
git add filename    #添加具体文件

    提交本地修改

    git commit -m "描述"

    拉取远程文件

    git pull originname originbranchname
